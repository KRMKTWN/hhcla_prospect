# HHCLA Prospect

Animated portrait presentation for Homeless Health Care Los Angeles and the proposed **>[ HUB ]> TEXTILE** program.

## Presentation structure

1. Official HHCLA identity
2. Prospect presentation
3. Our history
4. What we already know how to do
5. Campus → Workshops → Work Program → HUB
6. >[ HUB ]> TEXTILE
7. Existing operational proof and outcomes

## Purpose

The presentation introduces the HUB as infrastructure that organizes useful resources, paid participation, practical learning, repair, maintenance, and distribution across HHCLA community spaces.

The Textile branch is the first visible application of this system, connecting clothing recovery, sorting, repair, redistribution, dignity, and workforce development.
